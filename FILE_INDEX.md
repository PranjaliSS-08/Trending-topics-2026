## 📖 File Descriptions

### 1️⃣ app.py (850+ lines)

**Purpose**: Main Streamlit application

**Key Components**:
- Page configuration and styling
- Home page (project overview)
- Trend Analytics page (EDA with 8+ charts)
- Real-time Forecasting page (predictions)
- Utility functions for predictions
- Model loading and caching
- Error handling

**Main Functions**:
- `load_models()`: Load pre-trained models
- `load_dataset()`: Load CSV data
- `predict_trend()`: Make predictions
- `page_home()`: Home page UI
- `page_trend_analytics()`: Analytics page UI
- `page_forecasting()`: Forecasting page UI

**Run with**: `streamlit run app.py`

---

### 2️⃣ train_model.py (280+ lines)

**Purpose**: Train and save ML models

**Key Components**:
- Data loading and preprocessing
- TF-IDF vectorization
- Feature engineering (text + categorical)
- Random Forest model training
- Logistic Regression model training
- Model evaluation metrics
- Model persistence

**Main Functions**:
- `load_data()`: Load CSV
- `preprocess_data()`: Encode categories
- `create_vectorizer()`: TF-IDF setup
- `train_engagement_model()`: Random Forest
- `train_sentiment_model()`: Logistic Regression
- `save_models()`: Pickle models

**Run with**: `python train_model.py`

**Output**: Creates `models.pkl` (required for app.py)

---

### 3️⃣ trending_topics_2026.csv

**Statistics**:

- Regions: 3 (Global, India, USA)
- Engagement: 0-100 scale
- Sentiment: 3 classes (Positive, Neutral, Negative)
